# Icon Creation Guide

## Requirements
- **Size**: 128x128 pixels
- **Format**: PNG
- **Location**: `/Users/quercy/Downloads/DoNotDisturb++/DD/icon.png`
- **Name**: `icon.png` (exactly)

## Quick Options

### Option 1: Online Tools
1. Visit [Favicon.io](https://favicon.io/favicon-generator/) or [Canva](https://www.canva.com/)
2. Create a 128x128 image
3. Use a bell icon, "DND" text, or notification symbol
4. Download as PNG
5. Save as `icon.png` in the extension root

### Option 2: Use VS Code Icon
1. Find a bell or notification icon online
2. Resize to 128x128 using any image editor
3. Save as PNG

### Option 3: Simple Text Icon
1. Create a simple design with:
   - Background: Dark color (e.g., #1e1e1e)
   - Icon: Bell or "DND" text in white
   - Border: Subtle border
2. Export as 128x128 PNG

## Design Suggestions
- Use a bell icon (ðŸ””) to represent notifications
- Use "DND" text (Do Not Disturb)
- Use a crossed-out bell or notification symbol
- Keep it simple and recognizable at small sizes

## After Creating
Once you have `icon.png` in the root directory, you're ready to package!
